# TabletHack
把学校的学习平板变成浏览器！（（

利用电教老师提供的 GeoGebra 软件可以浏览网页，只需用安卓手机下载 [TabletHack 传送门](https://www.geogebra.org/m/ptttsuur)并用蓝牙传输给平板。

顺带一提，这个 README 文件也是使用平板 TabletHack 编辑的！（
